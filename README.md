## 🎮 روش استفاده

1. IP ها رو وارد کن (یکی در هر خط) یا Paste کن
2. Mode انتخاب کن:
   - **⚡ Simple** — سریع، SNI: google.com
   - **🧠 Auto-SNI** — پیشرفته، CDN detect + همه SNI ها
3. **START SCAN** بزن
4. نتایج رو **Copy** یا **Save** کن
5. بذار توی شیر خورشید:
   ```
   Settings → CDN Fronting → CDN edge IPs
   ```
